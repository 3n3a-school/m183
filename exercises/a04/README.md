# Oauth2 and OpenId Connect Example

## Usage

To start using this one must fill out the provided `sample.env` and rename it to `.env`.

## Available Scripts

In the project directory, you can run:

### `npm run dev`

To start the app in dev mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm start`

For production mode

### `npm run test`

Run the test cases.

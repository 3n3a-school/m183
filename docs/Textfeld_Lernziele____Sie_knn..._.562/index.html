Lernziele:

  • Sie können die gegenwärtige Situation des Internets hinsichtlich Nodes (Geräte), Traffic, Netzwerk, Protokolle, und Applikations-Architektur aufzeigen und bewerten.
  • Sie können die beiden Begriffe Application Security und Web Application Security vor dem Hintergrund der gegenwärtigen Situation des Internets beschreiben.
  • Sie kennen verschiedene generelle Prinzipien der Applikationssicherheit und können diese konkret beschreiben.
  • Sie können die Denk- und Vorgehensweise eines Hackers gemäss den Unterrichtsinhalten erklären und deren Bedeutung für die Applikationssicherheit aufzeigen.
